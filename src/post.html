Модуль «Спойлери»

категория: Документация «Мечта верстальщика»
@@include('html/_widgets.htm',{})

Спойлер – это заголовок, при нажатии на который под ним разворачивается определенный контент.

Основные возможности

  1. Использование множества блоков со спойлерами
  2. Выключение/включение функционала на определенной ширине экрана
  3. Функция «аккордеон», когда в блоке может быть открыт только один спойлер
  4. Возможность анимированного открытия
  5. Семантик

Подключение функционала

[HTML] В нужном месте вызвать снипет spollers (классы можно заменить нужными). Или вручную создать структуру с соответствующими дата-атрибутами. Обратите внимание, что добавление класса _spoller-active для элемента с атрибутом data-spoller сделает спойлер активным (открытым).

Пример блока с двумя спойлерами:

								<a href="#" data-goto=".имя класса блока" class="link">Пункт навігацііa>

									<a href="#" data-goto="#id блока" class="link">Пункт навігацііa>
